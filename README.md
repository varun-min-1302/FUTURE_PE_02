# 🚀 AI UGC Ad Generator (Task 2)

An AI-powered system designed to generate high-converting, authentic UGC (User Generated Content) ad scripts using structured prompt engineering.

---

## 📌 Overview

This project is part of the Prompt Engineering Internship at Future Interns.

The goal was not just to generate content, but to build a **scalable system** that produces UGC-style ads similar to those used by real D2C brands and marketing agencies.

These ads are optimized for:
- Instagram Reels  
- YouTube Shorts  
- Meta Ads (Facebook & Instagram)

## 🏷️ Product Chosen

**Minimalist 10% Niacinamide Face Serum**

- Audience: Men & Women (18–35)  
- Market: D2C Skincare (India)  
- Price: ₹399

## 🎯 Objective

To design a reusable AI prompt system that can:

- Generate scroll-stopping hooks  
- Create realistic UGC-style ad scripts  
- Produce high-converting CTAs  
- Generate platform-specific captions  
- Adapt content for multiple brands  

---

## ⚙️ System Workflow

Input → Prompt → AI Output → Refinement → Final Ad Content

1. Define product details and audience  
2. Apply structured prompts  
3. Generate multiple content variations  
4. Refine for authenticity and clarity  
5. Use output for real-world ad creatives  

---

## 🧠 Prompt Engineering Strategy

This system is built using advanced prompt engineering techniques:

- **Role-based prompting** (UGC creator + marketer mindset)  
- **Structured outputs** (Hook → Problem → Solution → Proof → CTA)  
- **Tone control** (casual, relatable, non-salesy)  
- **Context injection** (brand voice, audience, product details)  
- **Modular design** (separate prompts for hooks, scripts, CTAs, captions)  
- **Reusability** across different products and industries

## 🧪 Hook Testing Strategy

To maximize performance, multiple hook variations are generated using different psychological triggers:

- Fear (skin issues, acne, oiliness)  
- Curiosity  
- Social proof  
- Comparison  
- Personal storytelling  

Each hook can be tested independently to improve:

- First 3-second retention  
- Engagement rate  
- Conversion rate
